<h2 id="-english">English</h2>

**Retro Pod** is a Chrome Extension that introduces an **iPod Classic interface** to Spotify via the modern **Document Picture-in-Picture API**.
It opens a floating, always-on-top window that lets you control your music and view synced lyrics while you work.

### Features

* **Document Picture-in-Picture (PiP)**
    Adds a floating iPod window that stays on top of other apps.
* **Functional Click Wheel**
    Scroll to change volume, click buttons to control playback.
* **Synced Lyrics**
    Beautiful, scrolling lyrics that sync perfectly with the music (Apple Music Sing style).
* **Multiple Themes**
    Silver, Black, and the **U2 Edition** (Black & Red).
* **Menu System**
    Toggle Shuffle and Repeat directly from the iPod interface.
* **Native Volume Control**
    Hacks Spotify's volume slider for precise control via the scroll wheel.

### Installation

Since this is a proof-of-concept project, please install it manually:

1.  **Clone or Download** this repository.
2.  Open **Chrome** (or Edge/Brave) and navigate to `chrome://extensions/`.
3.  Toggle **Developer Mode** in the top right corner.
4.  Click **Load Unpacked**.
5.  Select the folder containing this extension's files.
6.  Open [Spotify Web Player](https://open.spotify.com) and click the floating **"iPod Mode"** button.

### Controls

| Interaction | Action |
| :--- | :--- |
| **MENU Button** | Open / Close the Menu |
| **Center Button** | Select Item / Toggle Lyrics Mode |
| **Wheel Scroll** | Adjust Volume / Navigate Menu |
| **Wheel Buttons** | Play/Pause, Next, Previous |

<h2 id="-japanese">日本語</h2>

**Retro Pod** は、最新の **Document Picture-in-Picture API** を使用して、Spotify Web Playerに **iPod Classic風のPiPウィンドウ** を追加するChrome拡張機能です。
最前面に固定されたウィンドウで、作業をしながら歌詞を見たり、クリックホイールで音量を調整したりできます。

### 主な機能

* **ピクチャー・イン・ピクチャー (PiP)**
    常に最前面に表示されるウィンドウを追加します。
* **動くクリックホイール**
    マウスホイールのスクロールで音量調整やメニュー操作が可能。
* **歌詞同期 (Synced Lyrics)**
    Apple Music Singのような美しい歌詞スクロール表示。
* **テーマ変更**
    Silver、Black、そして **U2 Edition**（赤黒モデル）。
* **メニュー操作**
    シャッフルやリピートの切り替えもiPodの画面から操作可能。

### インストール方法

Chromeウェブストアには公開していないため、以下の手順で手動インストールしてください：

1.  このリポジトリを **Clone** または **Download ZIP** して解凍します。
2.  Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の **「デベロッパーモード」** をオンにします。
4.  **「パッケージ化されていない拡張機能を読み込む」** をクリックします。
5.  解凍したフォルダを選択します。
6.  [Spotify Web Player](https://open.spotify.com) を開き、画面右下に出現する **"iPod Mode"** ボタンを押してください。

### 操作方法

| 操作 | 動作 |
| :--- | :--- |
| **MENU ボタン** | メニューを開く / 戻る |
| **センターボタン** | 決定 / 歌詞モードの切り替え |
| **ホイールスクロール** | 音量調整 / メニュー移動 |
| **左右ボタン** | 前の曲 / 次の曲 |
